---proposal

Placing yourself into a certain environment encourages an awakening of specific subconscious memories and feelings that help to recall a certain knowledge as well as influences a formation of new ideas.

Flexible multi room dimension (FMD) is an embedded model of memories storage and retrieval mechanism, which allows a person to recall and access memories, store moments, think, dream and create ideas with spatial interaction with environment.Through a multiple connections, interactions, and crossaccosiations, the person can place themselves in a memory room and, by surrounding themselves with different environments, FMD help to recall details of dependent taxonomies or, by mixing a certain memories details, produce new imagination environment.

Major Functions and Interactions:

  • store memory (room creation) - new memory will be created as an object with attributes and behaviors. Based this to be applied, memory will be placed in a specific room, or, in case if memory is very new, it may create a new room.
  • memories retrieval (retreival journey) - FDM helps to retrieve some memories that are not constantly clear by default state. It allows to open a gate or set of gates through association connections and place a person into the situation for easir details access.
  • idea creation (imaginarium) - a special room of dreaming. By selecting association aspects and connections, the person allows stored memory information to reach Imaginarium. Through mixing aspects in different amounts, new dreams and ideas got created.
  • fast access (house reshape) - the house can be reshaped for allowing faster and efficient transition between spaces.
  • maintenance (house state) - set of FDM that allows constant access and circulation of certain type of memories. It places a person in a specific situation that allows access to particular knowledge and memories.

---organizational_shemes

Reorganization and House states

Depending on the purpose and request, the house can be reshaped for allowing faster and efficient transition between spaces: connections will shrink, be blocked or opened depending on input (question/request). Reorganization of the house is based on classification and results in a house state (current house state). House state affects goals/mood and behavioral abilities of the person. Depending on which state and organization is active, some memory details has more connections to the current state than others, which maintain a certain background (present). For example, if person is focused on work, the house state will be reshaped in a way when the access to work related information and skills can be accessed with festest ways possible.

Single Room contains set of memories organized in a particular way through conditioning: need and request (input) of the person defines wich memories and memory details need to be primary, and wich ones can have a secondary role at the moment.

Organizational Structure

FMD has a mixed structure: overall it is a granular multilevel matrix, but there is a possibility of hierarchically organised branches and subbranches on more detailed level of the memory details in rooms. It allows FMD easily reorganize the order and shapei(be reshaped for short or long term) depending on need, association and connections: multiple cross-connected rooms contain sets of memory details with related tags. Memory details to have internal and external connection with different distance.

Transitions and journeys

Different states of the house allow a variety of transitions from one memory room to another. Depending on type of request and distance between memory rooms and details, transition between the rooms can differ in time and experience (in other words, it can be direct and indirect). Sometimes accessing a certain memory can require multiple transitions. Besides that, FDM can generate a number of travel maps, that allows a person take different pathways while they think and discover different information through easier memory details access.

The reason of accessing the memories in some ways shapes a type of experience that the person may have with FDM. Single Room contains set of memories organized in a particular order. Memories may be rearranged based on priority of information type and request/need or priority association. There also a set of knowledge and memories that are needed in everyday life. This Key Memories carried with a person all the time

Connections

Memory details will be connected with related ones through different type of connection gates. Connection gates vary depending on type of relationships between memory details: they can have different length, diameter and material depending on type of connection and type of information. Each connection has maximum and minimum length, which directly affects ways how room can be reshaped. Besides, changing in connection length affects a clarity of a certain memories in the specific moment/state.

Connection gate types:

  • internal and external gatess are the connections between memories either within the room or outside of the room.
  • subconscious gates are the connections between memories that cannot be easily resized or blocked.
  • logical gates are the connections that are made through logical relations or associations.

---controlled_vocabulary

  • Room of Memory - accumulation of memories of certain cetegory, mostly based on general concept or topic
  • Memory detail or Association Aspect - type of information that forms memory and perception of the memories
  • Connection gate - connection that defines relationship type and strength between accociation aspects
  • Journey of type - traveling through memories and association aspects of specific type: emotions, knowledge, social experiences and etc.
  • Retrieval Journey - journey that allows to recall a specific siruation, detail, or set from the memory storage.
  • Travel Map - Map that shows a person possible journeys for recalling and restoring specific moments.
  • Transition - moving from one Memory Room to Another one
  • Default Knowledge Set - amount of memories that required for everyday experiences
  • House state - fixed shape of the FMD that opens a set of specific gates for maintaining retreval of srtain information.
  • Room Conditioning - tags and aspects applied to the room in order to maintain a certain atmosphere and environment
